Data

For this project I used the Spotify API to get the name of the artist and display the top 10 songs of the artist.

  • 'https://api.spotify.com/v1/search' - This API Endpoint gets the list of artist when the user searches for an artist
  • 'https://api.spotify.com/v1/artists/{id}' - This endpoint gets the artist info with the artist id
  • 'https://api.spotify.com/v1/artists/{id}/top-tracks'- This endpoint gets the artist's top ten songs
  • More api endpoints with details can be found at this cite https://developer.spotify.com/web-api/console/
